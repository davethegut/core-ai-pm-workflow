Skill Highlights

Deep Dives on Key Skills

These three skills show the depth of what's possible.

Doc Writer — 7 Formats, Auto-Detected

The AI infers the right document format from your request. Just describe what you need.

  • Concept One-Pager — new ideas, vision
  • Problem/Solution One-Pager — feature enhancements
  • Decision One-Pager — choosing between options
  • PRFAQ — big bets, launches
  • PRD / Structured Epic — full specs
  • Vision Epic — direction-setting
  • GitHub Issue — publish directly

Doc Reviewer — CTO and GM Personas

Not generic AI feedback. Based on Kent Beck's 3X framework with persona-specific critique.

CTO Review

Feature fit, simplicity, green field fallacy, phasing risk. Asks the hard engineering questions before your CTO does.

GM Review (Concept)

Phase clarity, problem gravity, adoption mechanics. Ensures your concept passes the "so what?" test.

GM Review (Decision)

Decision clarity, ownership, reversibility. Stress-tests your recommendation before it hits the exec room.

Morning Standup — Your Metrics, Your Way

Configure the metrics that matter to you. Works with any SQL-compatible data source.

Honest Assessment

Is This for You?

This is for you if...

  • You're a PM who spends too much time on docs, decks, and data pulls
  • You're comfortable running commands in a terminal (or willing to learn)
  • You want to onboard your team to AI-assisted workflows
  • You use GitHub for tracking work

This is NOT for you if...

  • You want a no-code, click-through tool
  • You don't have Claude Code or Cursor
  • You need enterprise-grade security review before using any tool
  • You're looking for a course to learn AI basics
