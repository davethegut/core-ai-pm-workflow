# Tasks

This directory is for your personal task tracking. Create these files as needed:

- `GOALS.md` — Your current priorities and quarterly goals
- `ACTIVE.md` — What you're actively working on (synced from GitHub via my-initiatives)
- `BACKLOG.md` — Ideas and future work
- `DONE.md` — Completed work archive

All task files are gitignored — they're personal to your workflow.
