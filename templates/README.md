# Templates

Document format templates used by the doc-writer skill. Add your own templates here.

## Included Templates

- `structured-epic-template.md` — PRD / Structured Epic format
- `vision-epic-template.md` — Vision Epic format
- `prfaq-template.md` — PRFAQ (Press Release / FAQ) format
- `one-pager-examples.md` — Examples of concept, problem/solution, and decision one-pagers
